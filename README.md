# 🚀 GenAI Sales Forecasting Dashboard

A powerful AI-powered Sales Forecasting and Business Intelligence Dashboard built using Python, Streamlit, Pandas, Plotly, and Machine Learning.

The application helps businesses analyze sales performance, identify trends, forecast future sales, and generate actionable business insights through an interactive dashboard.

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/GenAI-Sales-Forecasting-Dashboard.git
```

Navigate to the project folder:

```bash
cd GenAI-Sales-Forecasting-Dashboard
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
